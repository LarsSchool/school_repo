Om de website te openen is het startpunt /Sensorium_groep/index.html.
Dit html bestand moet je als eerst openen met de VSCode extensie live server (of five server, dat is een verbeterde variant.)
Door rechtermuisknop op het html bestand te klikken en dan op "open with five server" te klikken openen wij hem in onze browser.
Vanuit hier kan je alle pagina's bekijken. 

Mocht je het alsnog onoverzichtelijk vinden is er ook een navpage gemaakt, zodra je deze opent zijn er 4 knoppen met alle 4 de portfolio's gelinkt.
Lekker overzichtelijk en simpel dus!
