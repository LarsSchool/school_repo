Hieronder is te zien hoe mijn PCB schema eruit zag. Later is dit nog wat aangepast, maar hier heb ik helaas geen foto van. Je hebt bijvoorbeeld maar 1 relay side nodig, terwijl ik er 2 had.


Hier geldt hetzelfde als bij de foto hierboven, dit is nog het oude design met wat onnodige dingen.


Blijkbaar was er een limiet voor hoe breed je PCB mocht zijn voor het produceren ervan. Dit heb ik overschreden, omdat ik dacht dat er bedoeld werd dat dit een Eagle limiet was. Hierdoor is het ontwerp aangepast en mist nu de locatie voor de relais, maar dit heb ik op een andere manier opgelost. Ik heb namelijk in de bak zelf een houder gemaakt voor de relais naast de PCB.

Hieronder zijn wat foto's van de soldeerles en hoe mijn PCB eruitzag. Deze foto's zijn ongeveer halverwege het proces gemaakt.


Daarna heb ik het afgemaakt door alle andere stukjes er ook op te solderen.


Helaas was ik iets te enthousiast aan het solderen en had ik overal dingen op gesoldeerd, terwijl het voor sommige beter was om het direct op de PCB te solderen. Dit heb ik later thuis gedaan en hierna een testfit gedaan, waar deze foto uit voort is gekomen.
